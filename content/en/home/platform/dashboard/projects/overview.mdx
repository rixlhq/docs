Projects in **RIXL** are workspaces that organize your media, team members, and settings. Each project is isolated, giving you a clean separation of content, permissions, and analytics.

<Callout type="info">
  You need at least one project to use the main dashboard. If you don’t have one, you’ll be prompted to create it on first login.
</Callout>

## Getting Started

<Steps>
  <Step>
    **Create Project**

    Go to Settings → Projects → [Create New Project](/en/home/platform/dashboard/projects/create-project) to set up your first
    workspace.

  </Step>
  <Step>
    **Configure Settings**

    Adjust [Project Settings](/en/home/platform/dashboard/projects/project-settings) such as region, optimization, and access.

  </Step>
  <Step>
    **Add Team Members**

    Invite collaborators via [Team Management](/en/home/platform/account-teams/team-management).

  </Step>
  <Step>
    **Upload Media**

    Start adding [videos](/en/home/platform/dashboard/media/upload-videos) and
    [images](/en/home/platform/dashboard/media/upload-images).

  </Step>
</Steps>
