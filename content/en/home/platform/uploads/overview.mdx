## Supported Formats

**Images** _(converted to optimized AVIF)_

- **Common**: JPEG, PNG, WebP, GIF, BMP
- **Professional**: HEIC/HEIF, TIFF, JPEG 2000
- **Vector**: SVG (preserved as-is)

**Videos** _(transcoded for adaptive streaming)_

- **Input**: MP4, MOV, AVI, MKV, WebM, FLV
- **Output**: Adaptive HLS with multiple quality tiers
- **Thumbnails**: AVIF format with instant placeholders

**File Limits**

- Images: Up to 100MB each
- Videos: Up to 2GB each
- Bulk uploads: No limit on quantity
